## ✨Demo
You can visit template at [digikala.vercel.app](https://digikala-nextjs.vercel.app/).

## 🔧Technologies
* Next.js
* Typescript 
* Tailwind css
* Redux Toolkit
* Swiper 

## 🔥Features
* Using Redux toolkit as state manager
* Using Json server as database
* Fully responsive
* Mega menu
* Diverse sliders

## 📃Pages
* Main (index)
* Products List
* Product Details
* Cart
* Favorites
* Profile
* 404 Error(not found)

## 🚀Getting Started
1. Clone the project

  ```bash
  git clone https://github.com/MahmoudAzad/digikala-Nextjs.git
  ```

2. Install dependencies
  ```bash
  npm install
  ```

3. start development server
  ```bash
  npm run dev
  ```
